# Insider Threat Case Studies and Qualitative Coding Resources

## Overview

This repository contains a curated collection of publicly available insider threat case studies and the resources developed to support their structured qualitative analysis. The repository is intended to facilitate reproducible research on insider threats by providing a consistent framework for transforming unstructured incident reports into structured datasets.

The coding resources are designed for use by both human annotators and Large Language Models (LLMs) to support transparent, consistent, and scalable qualitative coding.

## Repository Contents

```
├── case_studies/        # Publicly available insider threat case studies
├── template/            # Coding template
├── schema/              # Coding schema
├── coding_guide/        # Coding guide with definitions and examples
└── README.md
```

### Case Studies

A collection of publicly available insider threat incident reports gathered from trusted sources. These reports serve as the primary data for qualitative analysis and dataset construction.

### Coding Template

The coding template defines the attributes used to analyse each insider threat incident, including:

- Actor characteristics
- Attack characteristics
- Organisational characteristics

The template provides a consistent structure for representing each case in a machine-readable format.

### Coding Schema

The coding schema specifies the predefined categories and allowable values for each attribute in the coding template. It promotes consistency and reduces subjectivity during the coding process.

### Coding Guide

The coding guide provides detailed instructions for assigning values to each attribute. It includes:

- Attribute definitions
- Category descriptions
- Coding rules
- Decision guidelines
- Worked examples

The guide is intended to support both human coders and LLM-assisted qualitative coding.

## Purpose

The resources in this repository support:

- Insider threat research
- Qualitative data analysis
- Structured dataset construction
- LLM-assisted qualitative coding
- Reproducible empirical software engineering research
